4.2 稀疏矩阵

稀疏矩阵(Sparse matrices)对象仅存储非零元素，更节省内存

Code
library("Matrix")
N = 100

m = diag(1, N, N)
sp = sparseMatrix(1:N, 1:N, x = 1)
 
object.size(m)
#> 80216 bytes
object.size(sp)
#> 3104 bytes

4.3 并行化

parallelization

Parallel computing

Code
library("parallel")
no_of_cores = detectCores()
no_of_cores
#> [1] 8

parLapply（cl， x， FUN， ...）

parApply（cl = NULL， X， MARGIN， FUN， ...）

parSapply（cl = NULL， X， FUN， ...， simplify = TRUE， USE.NAMES = TRUE)

makeCluster()此函数指定要使用的处理器数量

Code
library("parallel")


N = 10^4
sapply(1:N, sd)

cl = makeCluster(4)
parSapply(cl, 1:N, sd)
stopCluster(cl)
