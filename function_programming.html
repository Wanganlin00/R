4.2.1 Name masking

在函数内部定义的名称会掩盖在函数外部定义的名称。

Code 
x <- 10
y <- 20
g02 <- function() {
  x <- 1
  y <- 2
  c(x, y)
}
g02()
#> [1] 1 2

如果未在函数中定义名称，则 R 将向上查找。

Code 
x <- 2
g03 <- function() {
  y <- 1
  c(x, y)
}
g03()
#> [1] 2 1

y
#> [1] 20
