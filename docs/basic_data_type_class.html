1.1 数据类型

数据类型（mode）：表示对象在计算机内存中的存储形式

1.1.1 numeric – 数值型（integer/double）

integer – 整型 L

Code
mode(4.3)
#> [1] "numeric"
class(4.3)
#> [1] "numeric"
mode(1L)
#> [1] "numeric"
class(1L)
#> [1] "integer"

1.1.2 character – 字符型

Code
mode(c("car"))
#> [1] "character"
class("car")
#> [1] "character"

1.1.3 logical – 逻辑型

Code
mode(c(TRUE,FALSE))
#> [1] "logical"
class(c(TRUE,FALSE))
#> [1] "logical"

1.1.4 complex – 复数型

Code

mode(c(1+2i,3-4i)) 
#> [1] "complex"
class(c(1+2i,3-4i))
#> [1] "complex"

1.1.5 raw型

Code
charToRaw("abcde12345") #字符串中每个字符的原始存储格式（十六进制？）
#>  [1] 61 62 63 64 65 31 32 33 34 35
mode(charToRaw("abcde12345"))

#> [1] "raw"

1.2 数据结构

class：是一种基于面向对象的R的抽象类型划分,或者理解为一种数据结构。

1.2.1 vector

向量,同一类元素的集合。 ?c #Combine Values into a Vector or List

R中没有标量（scalar），只含一个元素的向量
Code
#单元素向量
1 == c(1)
#> [1] TRUE
is.vector(1)
#> [1] TRUE
  
"a"
#> [1] "a"
is.vector("a")
#> [1] TRUE

#使用函数c()创建向量

c(1)
#> [1] 1
c(1,2,3,4,5)
#> [1] 1 2 3 4 5
c("a","b","c")
#> [1] "a" "b" "c"

1.2.2 factor

因子，分类变量，其中每个级别都是一类。?factor factor(vector,ordered=FALSE,levels=c(v1,v2,…),labels= ,...) ，在内存中以整数向量c(1,2,3,...,k)存储。

Code
# 存储形式
class(factor((c(1,2,3)))) 
#> [1] "factor"
mode(factor((c(1,2,3)))) 
#> [1] "numeric"

# 名义变量 nominal variable
diabetes<-c("t1","t2","t1","t1") 
str(diabetes)
#>  chr [1:4] "t1" "t2" "t1" "t1"
diabetes<-factor(diabetes)
str(diabetes)
#>  Factor w/ 2 levels "t1","t2": 1 2 1 1

# 顺序变量 ordinal variable     默认水平根据字母顺序而定
status<-c("poor","better","best","poor")
status<-factor(status,ordered = TRUE) 
str(status) 
#>  Ord.factor w/ 3 levels "best"<"better"<..: 3 2 1 3
status<-factor(status,ordered =TRUE,levels = c("poor","better","best")) 
str(status) 
#>  Ord.factor w/ 3 levels "poor"<"better"<..: 1 2 3 1


#改变外在标签
sex<-c(1,2,2,1)
sex
#> [1] 1 2 2 1
sex<-factor(sex,levels=c(1,2),labels = c("男","女")) 
str(sex) 
#>  Factor w/ 2 levels "男","女": 1 2 2 1
sex
#> [1] 男 女 女 男
#> Levels: 男 女

leadership <- read_csv("data/leadership.txt") |> column_to_rownames(var = "id")
# 连续型变量→因子
leadership$agecat<-cut(leadership$age,
                    breaks = c(0,18,45,65,Inf),
                    labels=c("minor","young","middle_age","elder"),
                    include.lowest = TRUE,
                    right = TRUE
                    )
leadership$agecat
#> [1] young young young elder elder
#> Levels: minor young middle_age elder

日期/日期时间 默认格式："%Y-%m-%d" xxxx-xx-xx,例如：2023-03-15

1.2.4 matrix

矩阵，所有元素必须是同一类型。?matrix

matrix(data= ,nrow=1 ,ncol=1 ,byrow=FALSE ,dimnames=list(rnames,cnames) ,...)

Code
num<-c(16,22,24,28)
rnames<-c("R1","R2")
cnames<-c("C1","C2")
mymatrix<-matrix(num,nrow=2,ncol=2,byrow=TRUE,dimnames=list(rnames,cnames))
mymatrix   
#>    C1 C2
#> R1 16 22
#> R2 24 28

#> , , C2
#> 
#>    B1 B2 B3 B4

#> A3 15 18 21 24

1.2.6 data frame/tibble

数据框， 由列向量组成，每一列元素必须是同一类型，列与列之间长度必须相同，但类型可以不同。?data.frame

data.frame(name1=col_vector1,name2=col_vector2,...,row.names = ,...) ?tibble，tibble()是tidyverse风格的数据框，用法类似。

Code
id<-c(1,2,3,4) 
age<-c(21,14,52,15) 
diabetes<-c("t1","t2","t1","t1") 
status<-c("poor","better","best","poor") 
patient<-data.frame(patientID=id,age,diabetes,status,row.names = id) # 4个列向量组成数据框 
patient |> rownames_to_column(var = "ID")
ID patientID age diabetes status
1 1 21 t1 poor
2 2 14 t2 better
3 3 52 t1 best
4 4 15 t1 poor
Code
mode(patient)
#> [1] "list"
class(patient)
#> [1] "data.frame"
